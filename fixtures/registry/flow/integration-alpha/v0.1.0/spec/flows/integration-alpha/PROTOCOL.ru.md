# Протокол integration-alpha (русская версия)

Фикстура integration-alpha проверяет все surface'ы PROP-003 r2 в
комбинации: features, subskills, языковые предпочтения, условные
зависимости и связку через PURL `describes`.

Это русская версия. Канонический английский лежит рядом в
`PROTOCOL.md`; `vibe install` без `--language ru` подбирает именно
его через i18n fallback chain.
